Damage incident appears to be more likely at lower temperatures.

Aim is to model probability of damage at different temperatures, $P(t)$. Can use the logistic function for this:

$$P(t)=\frac{1}{1+e^{-\beta t+\alpha}}$$

where $\beta$ controls the slope and $\alpha$ shifts curve left/right (bias)

temperature = df['Temperature']
damaged = df['Damage Incident']

with pm.Model() as model:
    
    # Priors
    # Init both to 0 so P(t) has a reasonable starting value at 0.5
    beta = pm.Normal('beta', mu=0, sd=1000, testval=0)
    alpha = pm.Normal('alpha', mu=0, sd=1000, testval=0)
    
    # P(t) assumed to follow logistic function
    # Use Theano exp() to build this as compute graph
    p = pm.Deterministic('p', 1.0/(1.0 + tt.exp(-beta*temperature + alpha)))
    
    # Observed
    obs = pm.Bernoulli('obs', p, observed=damaged)

In [94]:
def logistic(x, beta=1, alpha=0):
    return 1.0 / (1.0 + np.exp(np.dot(-beta, x) + alpha))
